# Changelog

All notable changes to Orya One RaceSim will be documented in this file.

The format is based on Keep a Changelog and the project follows semantic versioning in a lightweight, practical way.

## [0.1.0] - 2026-03-22

First public showcase release.

### Added

- Next.js simulator workspace with premium motorsport-inspired UI
- FastAPI backend for defaults, strategy suggestions, and simulation runs
- Python simulation core with pace model path, strategy engine, event engine, and Monte Carlo aggregation
- synthetic sample datasets and schema documentation
- architecture, methodology, roadmap, and demo documentation
- public repo templates, contribution guidance, and MIT license

### Improved

- scenario narratives, event-impact summaries, and confidence language
- strategy recommendations with clearer tradeoff framing
- release-focused README, quick start path, and demo capture guidance

### Notes

- API tests run when FastAPI dependencies are installed; otherwise the API test module skips cleanly in bare environments
